# Harbr Screen Redesign

Redesign one screen at a time with a product-minded workflow.

## Process
1. Identify the screen's main user job.
2. Identify clutter, weak hierarchy, wasted space, and confusing actions.
3. Propose a better layout with clear structure:
   - top summary
   - key actions
   - primary content
   - secondary metadata
4. Implement the redesign in Flutter.
5. Preserve existing functionality unless explicitly changing behavior.
6. Check loading, empty, offline, and error states.

## UX principles
- One obvious primary action.
- High signal-to-noise ratio.
- Fast scannability for large media libraries and queues.
- Good thumb reach and tap targets.
- Avoid over-decoration.

## Deliverable format
- Brief audit
- Redesign plan
- Files changed
- Follow-up opportunities
