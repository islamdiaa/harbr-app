# Harbr QA Polish

Run a final quality pass on Harbr UI changes.

## Check
- visual consistency
- spacing consistency
- alignment
- typography hierarchy
- icon consistency
- tap target size
- dark mode / AMOLED behavior
- long titles and overflow
- loading / empty / error / offline states
- responsiveness on narrow mobile screens
- obvious accessibility issues
- regressions in navigation or action placement

## Output
- List issues found
- Fix the important ones directly
- Summarize any remaining non-blockers
